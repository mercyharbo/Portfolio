# CodeWithMercy Blog Roadmap

This roadmap focuses on high-quality evergreen posts that support the CodeWithMercy brand: frontend development, React, Next.js, TypeScript, workspace setup, AI for developers, career growth, product building, and practical software engineering.

Priority scale:

- High: Publish early.
- Medium: Publish after core clusters are started.
- Low: Useful later or when there is supporting project proof.

Difficulty scale:

- Low: Straightforward article.
- Medium: Requires examples or deeper explanation.
- High: Requires code samples, screenshots, benchmarks, or careful technical depth.
